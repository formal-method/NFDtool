## 项目简介

### 项目定位

数据库原理教学工具（数据库设计辅助工具）

### 项目特点

自动模式分解

### 软件功能

- 计算闭包

  + 计算增广集
  + 计算传递依赖
  + 计算闭包

- 计算关键字

  + 计算主码
  + 计算候选码

- 计算最小依赖集

  + 依赖分解
  + 去除多余依赖
  + 去除多余属性
  + 计算最小依赖集

- 模式分解

  + 分解为2NF
  + 分解为3NF
  + 分解为BCNF
  + 分解过程展示
